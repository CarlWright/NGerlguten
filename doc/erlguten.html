ErlGuten - version 2.1

ERlGuten is a batch system for producing PDF from XML.

Erlguten is free software.

The word ErlGuten is built from Erlang plus Gutenberg.

The goal of ErlGuten is to produce typographic quality output - this means that we aim at the highest quality possible for the typeset text. The ErlGuten text justification algorithms produce paragraph-optimized text with fully kerned data, normally the production of such text cannot be done by everyday word-processing programs but requires the use of dedicated typesetting programs.

Download

The transformation software is written in Erlang. To run the system you will need to install Erlang, which is available from www.erlang.org - you will also need to install the distribution.

Release information

This release is the first alpha release of ErlGuten. The alpha status indicates that we expect the ErlGuten system to change rapidly within the next few years.

The principal reasons for releasing the system are to:

  • Create feedback - I'd like to know what features to be included in future versions, I'd also like you to use the system and report any formatting errors.
  • Attract volunteers. If you are an expert Erlang programmer, are interested in typography and want to help, please contact me. If you have expert knowledge of layout, or of the internal details of different font files and want to help please contact us.

Future releases

Future releases of ErlGuten will address the following problems:

  • Font embedding - so far we can only use the PDF embedded type 1 postscript fonts - I'd like to extend this to cover all type 1 postscript fonts, together with truetype fonts.
  • A content management system for the production of digital newspapers.
  • Plug-ins. I will make a plug in architecture - so effects like drop-paragraphs. etc will all be implemented through a standard plug-in interface.
  • Viewers - I'd like to connect ErlGuten to the freetype project and make a GUI with the same interface as provided by the pdf module
