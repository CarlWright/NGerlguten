To run the demos, I do the following:

1. cd to the ebin directory.

2. Start up "erl".

3. Do "application:start(sasl).".

4. Do "application:start(erlguten).". This loads erlguten and starts the font_server and its 
   supervisor.

5. Do "{ok, PID} = erlguten:start_link()." to get an erlguten server and its PID.

6. Do "erlguten:format(PID, "../demos/test1.xml"). " to process the first example PDF input file. 
   It produces test1.pdf.

7. Do "erlguten:format(PID, "../demos/test2.xml"). " to process the second example PDF input file. 
   It produces test2.pdf.
